# 盤中零股 模擬下單練習系統

仿照「元大證券 App」盤中零股下單畫面做的 **模擬練習網站**。
讓不熟悉操作的人，可以在 **不花真錢、不會真的成交** 的情況下，反覆練習零股下單流程。

> ⚠️ **這是模擬練習，不是真的下單。** 所有委託、成交都是虛擬的，不會送到交易所、不涉及任何金錢，與元大證券無任何關係。

---

## 一、怎麼打開（最簡單）

電腦上：直接用滑鼠 **雙擊 `index.html`**，就會用瀏覽器打開。

手機上（安卓 / iPhone 都可以）任選一種：

### 方法 A：把檔案傳到手機打開
1. 把 `index.html` 傳到手機（LINE 傳給自己、AirDrop、雲端硬碟都行）。
2. 在手機上用 **Chrome（安卓）** 或 **Safari（iPhone）** 開啟該檔案。

### 方法 B：用同一個 Wi-Fi，手機連到電腦（推薦，可加到桌面變 App）
1. 在這個資料夾打開終端機（PowerShell），執行：
   ```powershell
   python -m http.server 3210
   ```
2. 查出電腦的區域網路 IP（PowerShell 執行 `ipconfig`，找 `IPv4 位址`，例如 `192.168.0.10`）。
3. 手機與電腦連 **同一個 Wi-Fi**，在手機瀏覽器輸入：
   `http://192.168.0.10:3210/`
4. **加到主畫面**：
   - iPhone Safari：分享鈕 → 加入主畫面
   - 安卓 Chrome：右上⋮ → 安裝應用程式 / 加到主畫面

   之後就像一個 App，全螢幕開啟。

> 想要有一個固定網址、隨時用手機開？可把這個資料夾上傳到免費的 GitHub Pages 或 Netlify，得到一個 `https://...` 網址即可。

## 二、操作流程（照著做）

1. 第一次進入會跳出 **注意事項**：勾選「我已完整閱讀」→ 按 **確定**。
2. 在「下單」頁設定 **價格**：
   - 點價格欄會跳出數字鍵盤，或用 **跌停 / 平盤 / 漲停** 快捷鍵。
   - 也可以直接 **點上方五檔的價格** 帶入。
3. 設定 **數量**：點數量欄輸入，單位是 **1 股**（不是「張」）。
4. 按 **買進（紅）** 或 **賣出（綠）**。
5. 在跳出的「委託確認」視窗按 **確認送出**。
6. 到 **委託** 頁看狀態：**餘** = 還沒成交的股數、**成** = 已成交股數。
   - 想取消按 **刪改**。
7. 成交紀錄與持有部位在 **明細** 頁。

> 模擬會每隔幾秒「撮合」一次，幫你示範成交效果（真實盤中零股約每 1 分鐘撮合一次）。

## 三、給新手的重點

- **零股**：股票 1 張 = 1000 股。零股可以 **1 股起買**，小錢就能參與。
- **顏色（台股）**：<span>🔴 紅色 = 漲 / 買進</span>、<span>🟢 綠色 = 跌 / 賣出</span>（和美股相反）。
- **限價 ROD**：你指定一個價格，當日有效；盤中零股只能用這種。
- 想快點成交：**買就掛高一點、賣就掛低一點**。
- 右上角 **?** 隨時可看教學；教學裡有 **重設模擬** 可清空練習紀錄。
